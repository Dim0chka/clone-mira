# Клиентская часть

Главное, запуск фронта
`npm run dev`

## Основное
- чтобы нажать на стикер, нужно на тачпаде нажать двумя пальцами
- создание стикера: на доске сверху есть иконка стикера, кликаете и появляются стикеры
- сортировка. По времени и т.д.

## Технологии
Фронт написан Vue3. Основные моменты на ts. Стилизация - bootstrap 5

## Почему выбор пал на такие технологии?
Vue3 простой для использования и весит немного
Ts - это вообще базовый язык с типизацией
bootstrap, просто потому что им пользовался большую часть времени и знаю название класса.
